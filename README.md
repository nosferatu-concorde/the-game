# IF !AI THIS.GAME

A code-themed platformer built at the **Supercell AI Game Jam** (February 6-8, 2026). Navigate through 10 levels of traditional anti-intelligence with IF and Else -- no AI allowed.

## Play

https://nosferatu-concorde.itch.io/the-game-jam-game

## Run locally

```
git clone https://github.com/nosferatu-concorde/the-game
cd the-game
npm install
npm run dev
```

## Controls

- **Arrow keys** or **WASD** -- Move and jump
- **Space** -- Start game

## Tech stack

- [Phaser 3](https://phaser.io/) -- Game framework
- [Vite](https://vitejs.dev/) -- Build tool
- [Inkscape](https://inkscape.org/) -- Graphics
